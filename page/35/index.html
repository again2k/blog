我不喜欢董方卓的八大理由!

嗯, 是"不喜欢"而不是"讨厌", 用否定句式语气应该要弱点.
1, RP不够好. 国青克劳琛事件, 大家不要说我老咬住这件事不放, 其实听听比利时球迷对他的评价中也有他RP不好的部分, 不信看看这里.
2, 长的不够帅. 我承认牵强
3, 他是大连人, 是大连足球人, 我对大连人没偏见, 但全兴事件让我蒙上阴影, 大连人不要K我哈.
4, 他在曼联队, 会引来很多很多"只关注董方卓型伪曼联球迷".
    4.1, 这帮伪曼联球迷, 只看有董上场的比赛, 没董不看.
    4.2, 这帮伪曼联球迷, 为了让董有上场的可能, 巴不得曼联锋线状态差, 今天Saha受伤, 明天Rooney受伤, 这样小董就有机会.

5, 有董在曼联, 以后国内报道曼联的新闻就是成了董方卓新闻.
6, 以后弗格森随便一句话稍微提及到董, 国内媒体就会大肆渲染超级夸张地报道: 弗格森说: 董方卓XXXXXX.
7, 以后英超转播, 解说员就会围绕着董方卓说个不停, "快, 传啊, 传给董" "哎呀, 这么好的机会不传, C.罗太独了吧!" (这个觉得近期是不大可能了, Rooney, Larrson, Saha, Solskjaer, Smith, Rossi排着长队呢, 鸡肋联赛杯曼联又被淘汰了, 董的机会不多)
8, 面对中国足球, 我心理变态.

炮轰我吧! 我不喜欢董方卓!

Sunday, 31 December 2006

SkyLanD "2006年度总结"

每到年终, 感慨总是很多, 尤其是在大学三年半, 去年的12月31日, 应该是个周六, 当时我是看着曼联比赛跨向2006, 而今年, 我应该 是听着CRI English跨向2007了.
年终嘛, 在工作的人肯定少不了这个"年终总结", 我是学生, 但也有必要写写"年终总结", 但此"年终总结"非彼"年终总结 ".
先依靠我的回忆和Blog的Archives, 用简单话语记记流水账:

1月, 考完大三上学期期末考试, 然后得了大学以来最严重的重感冒, 瘫在床上一周, 又不能回家. 非常痛苦, 心理第一次痛恨北京. 同月底, 川足解散, 郁闷了好长时间, 疯狂地在网上找全兴的比赛录像, 但无奈全兴时代网络不发达, 网上没有. 最后这个月过了一个Happy的春节.
2月, 返校前彻底丢弃MP3, 买了个CD机, 从此音乐人生返回CD时代. 刚回学校, 开始思考是不是考研, 但只是思考没行动, 好友Wall4ever已经去教室占座去了. 这个月 Smith在足总杯受重伤, 那个画面看的另人心碎. 月底, 曼联夺联赛杯冠军.
3月, 仍然只是在思考考不考研, 和大学一好友关系彻底决裂, 我发过誓, 我们永远不可能恢复关系, 至少不可能恢复到以前那种状态! 这个月, 曼联努力的追着切尔西. 冠军杯中, Arsenal干掉我"第二讨厌"的Juventus. 小小地作了一小会枪手球迷.
4月, 初步决定考研, 也有所行动, 开始看高数了, 不过考哪里没决定. 和同学在外面租了一套房子, 正式搬出去住. 月底, 曼联客场0:3败给切尔西, Rooney重伤, 我流下伤心眼泪, 看着 Lampard和Terry站在一起关注着地上痛苦的Rooney, 我的心当时软了, 实在控制不住, 眼泪就流出来 了.
5月, 边期末复习, 变考研复习, 收到GGAD支票, 虽然收到了支票但并没有鼓励我继续搞这个, 这不, 现在的Blog已经没有Google的广告了. 英超结束, 曼联第二, 车子背靠背冠军, 历史第三次夺得联赛冠军, 次数是曼联的1/5, 利物浦的1/6, Barca欧冠夺冠. 欧洲足坛进入英西争霸. Keane从Celtic退役, 曼联传奇队长告别足坛!
6月, 期末考试和世界杯重叠, 幸好重叠的不多, 考完后, 决定暂别考研复习, 认真看世界杯, 关注英格兰, 天天骂Erickson, 怒英格兰队不争! 黄健翔发彪, 一群意大利球迷支持着老黄, 黄是什么第一解说? 说这种话的人绝对没听过ESPN的解说, 认识詹俊, 苏东等人么?
7月, 英格兰意料之中地被葡萄牙淘汰, 在Blog上失态了一番(那篇日志已经删了), 之后冷静下来总结了英格兰06之旅, C.罗在比赛中的举止没惹怒我, 但他说他想去我"第一讨厌"的皇马后, 让我对他第一次失望.  意大利夺冠, Zidane悲情告别, 我不像很多老球迷那样从小看意甲长大, 我是看英超长大, 所以我不是什么意大利球迷, 虽然我也不是法国球迷, 但真为Zidane和法国感到惋惜. 之后进暑假, 大多同学都回家了, 我选择了不回家, 在学校考研复习, 几个战友报了班, 暑假天天去上课, 我没报, 一个人生活着.
8月, 英超进入新赛季, 我也进入复习浮躁阶段, 买了件曼联新球衣, 希望曼联新赛季一切都好, 当我有点想家的时候, 同学陆续返校, 新学期开始了, 曼联开局很顺, 一切都在往好的方面发展.
9月, 曼联最终还是没买Torres, Tevez等人. 我也正式进入考研的三点一线生活, 但这时的企业宣讲会也不时而时地影响着我, 有迹象表明我的生活要改变了…
10月, 果然, 经过反复衡量思考, 放弃了准备了大半年的考研, 正式找工作. 重庆力帆降级, 小小幸灾乐祸了一把. 正式开始找工作, 面了几次试, 月底, 学校第一场招聘会, 先是备受打击, 之后终算签到了工作. FM2007发行, 进入CM/FM第7年. 又加入"红魔周刊"团队, 负责做封面, 嘿嘿.
11月, 考完本学期所有试, 提前进入"寒假", 由于期末有实习, 只得呆在学校. 正式转正, 成为正式共产党员. 天天呆在宿舍, 只要面对电脑, 我是不可能感到无聊的. 曼联在对车子的比赛中暴露了很多问题, 非常精彩和有意义的一战!
12月, 继续"宅男"生活, 偶尔出去走走, M$正式发行Vista, 硬件限制使我至今没装, 对Vista的认识停留在去年装了它第一个测试版的阶段. 保存了两年的QQ聊天纪录被误删, 着实伤心了好一阵子. 曼联租下Larrson, 欧冠抽到好签. Blog正式从Zblog转到Wordpress.

这一年大概就是这个样子, 有得有失, 依旧没什么大事, 如果真要找出大事件来, 放弃考研找到工作应该算.
感谢所有我要感谢的人!  (汗.. 怎么感觉是废话….) 大家元旦快乐, 2007年事事顺心!
明年7月我也要告别校园生活正式工作了, 2007年, 我充满期待!

感受WordPress的强大魅力 —- Z-Blog to WordPress

记不清曾经下过几次决心要把Blog的程序转换成WordPress了, 只记得每次都被”麻烦”两个字挡了回来, 怕转不好数据丢失, 怕搜索引擎的收录丢失, 怕对PHP的陌生. Finally, 在这闲的”发霉”的大四, 在这Z-Blog作者ZX冬眠的季节, 我终于下了恨心转Wordpress, 经过一天左右时间的努力, 终于把以前的Z-Blog转到Wordpress来了.

这次转换, 得感谢月光写的Z-Blog转MovableType插件, 有了这个, 就可以轻松地把Z-Blog的数据转到Wordpress上了, 除了Tags以外, 其它都还完整. 做完第一步后, 为了减少搜索引擎的收录损失, 我把Wordpress的日志URL结构改成了和Z-Blog一样, 也就是 domainname/post/postname.html, 然后再一篇一篇日志修改, 把文章缩略名修改为以前Z-Blog对应日志的文件名, 很累人的说, 幸好之前我已经”精简”掉大量Blog初期”垃圾”日志. 这样修改完后, 基本上就OKay了, 然后把Playmyself.com域名设置好, 叫King把DreamHost那边设置好, 就等着生效了. 在域名生效期间, 我借助skyland.welog.org这个二级域名, 对新的Wordpress作了深入了解和插件定制.

首先得选一个好看的主题模板, WordPress是全球化的, 成千上万的主题选死人, 最后选择了这个iqwolf制作的 Unnamed One , 算是一见钟情吧, 看上这个主题后就没再找其它主题了. 此主题缺少”曼联味”, 原本我要找个曼联主题的, 但我随意在Bar上添加了些曼联小球衣和队徽, 算是有点曼联味了吧.

WordPress的强大在于他的扩展性, 各式各样的插件能满足你对Blog的任何要求, 一天时间, 我装了如下插件:

WordPress Database Backup WP自带的插件, 用来备份数据, 可下载到本地, 可发送到自己的邮箱.
wp-cache 用来建立缓存页面, 减少对数据库的读取, 增加访问页面速度.(我还没体会出来-_-)
Admin Drop Down Menu 增加后台管理功能, “减少超级多的后台鼠标点击次数”, 必备!
Ultimate Tag Warrior 超强的Tags管理功能, 我不但修改了每一篇日志的缩略名, 还给每篇日志重新添加了Tag, 累啊!
Twilight AutoSave 撰写编辑文章时, 定时自动保存, 像我这种做宿舍的学生是必备的, 因为宿管晚上熄灯从来”莫名其秒”.
Google Sitemaps 方便Google的收录.
Quote Comment 增加引用评论功能, 骂架时找对人骂, 以免产生误会-_-.
中文 WordPress 工具箱 调出最新留言, 最新日志, 随机日志等Blog常用的东西.
Gravatars2 大头贴, 在评论回复的时候显示你的头像, 前提是要在Gravatar注册. (我的只有在Firefox下才能看见, IE下看不见不知道为什么)
CoolPlayer 支持几乎所在媒体文件的插件.
Audio player 小巧的MP3 在线Player, 我肯定用的着.

就这么多了, 我现在也只是Wordpress菜鸟, 好多地方都不懂, 其它我还看上好多插件, 但都因为不会安装而放弃, 所以我要继续研究Wordpress! 也希望更多的人加入到Wordpress行列来!

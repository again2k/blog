Wednesday, 27 June 2012

永远悲剧的英格兰

英格兰国家队真是世界足坛的一朵奇葩，抛开英国媒体的固有吹嘘，其实单独把英格兰的每名球员提出来都是有头有脸的，在英超踢的风声水起，技术玩的起，力量使的起，传控球俱佳，一进入国家队一个一个的就只会长传冲吊，闭上眼睛就能想象到英格兰在场上的踢法，后卫解围时一脚开向前场，Welbeck 或 Carroll 努力去争顶，顶不到球直接落入对方守门员手上，要是顶的到的话，Ashley Young 或 Rooney 就像一条狗一样去追，当然结果基本追不到。打法非常难看，场面也被压制的非常难看，技术糙，无节奏，中场没人调度，拿不住球，带球三步必失误！踢瑞典虽然比分很热闹进球很精彩，那也只是灵光一现。点球死已经很给面子了，给人一种“悲壮”的感觉，其实实际比赛中踢成 X 一样，我本来还想在英格兰赢后好好讽刺下 Buffon 让他回家看毛片的，结果英格兰真争气，又把我这个标准伪英格兰球迷堵了回去。

为什么这帮球员在英超欧冠中都牛逼的很，一进国家队就萎了？英超没冬歇期球员太累？球员“国家荣誉感”不强？英超外援太牛逼，把国内球员埋没了？首先我觉得，没有一支英超球队是以英格兰人为班底向国家队输出球员，导致球员在俱乐部和国家队完全两种不同风格。Juventus 之于意大利，Bayern 之于德国，Barca 之于西班牙，这些俱乐部豪门有大量的本国球员，而且俱乐部风格和国家队基本相同，这样就能形成一个厚实的基础，球员容易把状态和打法打进国家队，而看看英格兰，曼联？曼城？车子？枪手？他桶？热刺？这些豪门哪支是国家队专业户？哪支的战术风格同英格兰相似？都不是，甚至都不能把这些豪门同英格兰联系在一起，我在想，其实英格兰国家队应该去英超中下游甚至英冠去选拔球员，然后继续打法不变，可能还有点希望。

似乎英超和英格兰产生了矛盾？！当然在盛世英超和盛世英格兰两者选其一，我当然还是要选盛世英超的。

Linode 试用笔记

看到有人免费分享马上到期的 Linode VPS, 去要了一个，试用了一下传说中的 Linode, 现在我的 Blog 已经转移到了 Linode 美国 Fremont 机房，但毕竟这个 VPS 不会长久，所以过不了多久可能又要转回原来的虚拟空间，也可能转到另一个 VPS。已经搬到 BuyVM（现在这个 Blog 就拿来折腾吧，反正也不知道写些什么了。）

这个笔记本来是我准备扔到 Evernote 作备份记录的，但想了想还是再往 Blog 多扔一份吧，所以这只是我自己记录怎么傻瓜化操作 VPS 装个小 WordPress 的记录，不是什么教程，这种教程网上一大把。

胡乱搞一通，慢慢摸清基本用法后决定 Rebuild 整台 VPS

0, 选择好操作系统与配置！Ubuntu 12.04 LTS Disk: 20224 MB Swap: 256 MB RootPSW:
REBUILD 等待
1，Boot VPS 等待
2，Remote Access 中找到 SSH 相关信息，用 Tunnelier 登录 SSH 打开 Terminal Console
3，照 lnmp.org/install.html 配置好 LNMP 环境，其中 nzh.me 的某个二级域名绑定 VPS
4，等待脚本安装编译完所有组件，要较长时间
5，创建 BLOG 域名的虚拟主机，设置好网站根目录路径
6，登录旧空间 SSH 打包整站数据和 MySQL 数据库到某目录下
7，再登录 VPS SSH 把整站数据与 MySQL 下载到 VPS Blog 域名的根目录，解压整站数据，删除下载过来的整站数据包
8，登录 phpMyAdmin 创建新的数据库，在 VPS SSH 上导入原数据库，删除下载过来的数据库
9，通过 SFTP 的方式编辑新的 wp-config.php 文件
10，在 dnspod 上把 Blog 域名解析到 VPS 上来
11，Done!
12，待安装优化组件并测试

我对 VPS 要学的还很多，先不管它对一个小博客的优势比以前的虚拟空间大多少，至少可以通过弄 VPS 学会很多东西。

Friday, 8 June 2012

迎接欧锦赛

看的第一届大赛是法兰西之夏，记忆中最精彩最难忘的大赛，那时课本上画满了 France 98 的 Logo, 通过 Adidas 的广告和对阵哥伦比亚的任意球让我认识 Beckham 继而引领我关注在今天被我视为信仰的曼联。

之后的 02 06 10 世界杯再也没有任何足球之外的回忆点。

再说欧锦赛，2000 年荷兰比利时欧锦赛是我唯一比较完整地看过的欧锦赛了，初三中考结束后看足球比赛的感觉大家都懂的，我在网上另一名字 Again2000 的来源之一也是这届欧洲杯，由于开始追曼联，从而在我心中终于有了国家队支持对象，那就是英格兰。对葡萄牙那场经典当时正值中考前几天，第二天满街的人都在讨论 Figo 的 Bug 超远射。考完试的当夜，英格兰对德国，Beckham 任意球传中球弹地后 Shearer 头球，最终 1:0，小组最后一场 P.Neville 的失误送点，最终输给罗马尼亚小组未出线，12 年过去了，当时英格兰比赛的细节都还能清晰地回忆起来，第一次总是那么难忘，那一年，2000 年，我充满太多欢乐。

04 年欧锦赛，比较悲催，正值大一结束，补大一开学没进行的军训，军训的前几天赶上一场英格兰对法国，那记忆是苦涩的，Beckham 又悲剧了，英格兰又悲剧了，Rooney 这届杯赛后随即转会曼联。那年欧锦赛后面的比赛，全是依靠军训间期购买的体坛周报打发的。军训结束的当晚是葡萄牙与希腊的决赛，在学校一咖啡厅坐在地板上看完了希腊神话，然后也倒在地板上睡了一晚。

08 年欧锦赛我已经工作一年，比赛期间又是正值培训，悲催的我只看了一场小组赛直播，连西班牙对德国的决赛都没看。

这届欧洲杯我本来也想像 10 年世界杯一样休一个月长假在家爽爽地看，但无奈最近太忙只得作罢。现在大家对欧洲杯的期待比世界杯高，如果这届欧洲杯还像 10 年世界杯那样无趣的话，那么请 FIFA 和 UEFA 改革吧！

有了 BloodZeed 不看广东体育好多年

还在找哪看英超吗？是的，什么新浪、搜狐、PPTV 都有英超直播，但这些大型门户为了“照顾”大多球迷的网络情况，都是些极其不清晰的视频，他们所谓的高清完全就是垃圾，其中搜狐的英超直播是最好的，但是比起今天介绍的 SopCast 上的 BloodZeed 来简直弱的批爆，甚至直接“被我”秒杀了电视上的广东体育。

首先，这个超高清直播服务是基于 SopCast 上的，可以到基官网下载，完全免费，直接不用注册登录，要看 Bloodzeed HD 的 Stream，直接在地址栏输入 sop://broker.sopcast.com:3912/106723，[2012-2013 赛季地址换成： sop://broker.sopcast.com:3912/132927]，或你装了软件后直接点这个链接即可，当然，没有比赛的日子，这个频道是停止的，到了英超比赛时间大概半个小时前，就有媒体流了，豪门的比赛都有。这个是超高清视频流，所以需要的带宽也是要很大的，据我测试 4M ADSL 足够了。Bloodzeed 是个狂热的热刺球迷，FM 玩家，他的推是 @Bloodzeed, 可以 Follow 他获取转播信息，他做流的台一般是 ESPN HD, SkySports HD 等。

当然，国内的哇哈直播也在做流，主要是 ESPN 亚洲和卫视台湾的直播，以前他们也有类似 Bloodzeed 似的超高清的，但现在没了，他列出的 ESPN 亚洲 HD 也不是超高清，但比国内那些强多了，更重要的是，可以听桂斌、詹俊的解说。我一般看非曼联比赛或需要点带宽时会选择这个。

收藏以上列表，高清在线享受英超，享受国外解说，享受 ESPN 桂斌詹俊们的解说，国内的那些就算了吧！

域名转到 GoDaddy

本 Blog 的域名是 05 年在新网互联的一个代理处注册的，在当时备案风声还没有那么紧的时候简简单单地就完成了域名备案，Blog 经过无数折腾，空间换来换去，前段时间新网互联老发邮件催我备案，我就想我不是备案过了吗，于是去工信部查询，发现已经查询不到我的备案信息了，于是就想过把域名转出到国外，在网上搜索了一些文章发现域名转出比较麻烦，于是还是想主动备案，我天真地以为现在备案还是像 05 年那时那么简单，结果我错了，现在备案还真麻烦，完全不能直接在网上操作，各种资料需要填写邮寄。域名转出和备案两者我还是选了域名转出。

虽然新网互联在域名管理界面开通了在线获取转移码的功能，但我是通过代理买的域名，还得要他开通在线转移功能才行。在 QQ 上好不容易找到新网互联那个代理，跟他说明我想转出后，他先用各种话语安慰我别转出后，我直接态度坚硬地表明转出的立场，终于他发了一份类似免责表给我，让我打印出来，签名后再扫描成电子档直接发给新网互联，虽然里面还要附加身份证扫描件等让我不爽的条件，但我还是忍了。搞定后发过去，就开始了漫长的等待着发给我转移码，经过 N 久后还没有消息，于是联系代理询问，他只回复要等上级回应，早在网上听说过国内转移域名遇到的种种阻碍，我感到一丝不安，不和不已要用网友的“杀手锏”了——直接投诉，我先是直接向新网互联的客服发了邮件，然后参照网上的“投诉体”直接给 ICANN 写了封投诉信，又开始第二轮等待，没过几天，我在登录域名管理后台，发现在线转移开通了（但期间还是没收到在线转移开通的邮件，幸好我自己亲自登录后台去看开通了没。看来都到最后了，他们还想阻止我转出啊）。现在一切简单了，成功拿到转移码后，直接去 GoDaddy 登记转入域名，成功登记后开始等待新网互联的放行，一般情况下被转出方不处理的话默认 5 天后自动放行，但还是担心地等待了几天后再到查 WhoIS 查询发现终于转移成功。这样域名空间全部脱离天朝，其实我的 Blog 内容健康和谐，没必要全部移民的，但我个人心理不能接受种种规定，还是出去放心。

趁优惠码，又继续在 GoDaddy 上注册了几个米，介于它的 DNS 服务器老被墙，这堆域名的 DNS 服务器全换成 DNSPod, 很好用啊！nzh.me 暂时作为个人主页索引，footballog.com 暂时没想好做啥，先搭个 WordPress 稍微采集了点东西先养着，等灵感来了有空了再来做，但我是估计这个域名一年后就直接荒废了，因为实在没时间和精力去搞了。

webOS HP Veer 试用三周小心得

话说 Nexus S 卖给印度阿三后，急需另一部手机替代，要求价格便宜，能折腾，纵观当今手机几大阵营，iOS, Windows Phone, Android 全都用过了，貌似还差那个被 HP 抛弃的 webOS 没用过, 加上 Veer 便宜的价格，于是就入了个 HP Veer. 本来其中“玩一玩”的心态占很大比重，但玩了两三周后，深深喜欢上了这个小东西，以后一段时间，它基本“主力”了。

小是我对 Veer 的第一印象，现在我管它叫做“陀陀儿”手机，屏幕只有 2.6 寸，在当今来不来就上 4.0 的手机当中非常另类，整个手机直接可以摊在手上，就算把键盘划出来也还是小巧，加上圆润的造型，其实更适合女生用，通过这几周的使用，虽然小，但用起来还是很舒服的，就是在公交车上看手机太吃力，容易头晕。整个触摸屏用起来滑动的感觉非常顺畅，加上 webOS 独有的手势，滑起来真的有指尖跳舞的感觉，什么 iOS, Android 真是弱爆了，以至于再用其它手机时总是习惯去找手势区划。向下滑出来的物理 Qwerty 键盘也很小，基本只能靠指甲按，好在手感不错，输入上还算流畅，值得一提的是，抽出键盘和合上键盘的手感，太爽了，以至于现在我没事就喜欢把键盘滑出来合回去。

由于汉化系统不太稳定，所以还是选择了 Stock Rom, Veer 绝对满足我“折腾”的需求，由于 webOS 已经被抛弃，导致现在的功能还是很弱，不得不依靠 Patch 来解决很多问题，虽然貌似很麻烦，但我却乐于折腾这种“定制”的快感，虽然稍微不注意造成 Patch 冲突，但通过不断实践，折腾出自己的一套 Patch 装备还是很有成就感的，刚开始的一周内，我不断打补丁，机子也不断重新刷机，现在终于稳定下来了。

webOS 另一比较弱势方面就是 Apps 了，由于 HP 已经放弃，导致现在的应用方面都停了下来，很多软件都是一年前的了，现在基本全靠纯 webOS, Palm 的爱好者们还在开发软件，虽然 Apps 很旧，但很多软件质量绝对可以和 iOS 媲美，更别提那些 Android 的了，举个例子就是，国人开发的 webOS 新浪微博是我用过所有平台中最好的微博客户端了，谁用谁知道。现在我需要的 Apps 基本能找到，Twitter 客户端的 Carbon 也是好用的很，翻墙方面由于 webOS 不支持 L2TP 等 VPN，但好在 IGVPN 提供的 VPN 有专门支持 webOS 的服务，所以翻墙出去也非常方便。

Veer 不爽的一些地方有：1，联系人，最开始我是添加 Google 账号同步联系人的，但由于天生对中文的支持不好，中文联系人不支持拼音搜索，而且同步方面也很恼火，只支持单向；2，日历，添加的 Google 日历也是基本不能同步，就算装了对应的所谓能解决问题的 Patch，但效果还是不理想，奇怪的是如果用 Exchange 添加 Windows Live 的账号，联系人和日历同步都表现的完美。好在 webOS 可以添加不同账号并设置不同的同步对象，加上我以前用 Windows Phone 手机时建立了完善的 Live 资料，所以我现在用添加 Live 账号用于同步联系人和日历，添加 Google 账号用以收发同步邮件和 GTalk, 这方面算是完美了。

总之，HP Veer 基本满足了我现阶段对手机 “小众”“能折腾”“便宜”“够用”的需求，在用的过程中其实也明白这手机真心不是普通人能用的，当然也完全明白了 webOS 为什么会“死去”。 最后扔图 + 简单点评：

毕业以后，只买过一张 CD, 更别说听 CD 了，曾经在中国政法大学南门淘 CD 的激情早就没有了，最近玩 FM2008 热刺的时候，盯着电脑机箱上的刻录光驱，盯着床头那台房东的 Panasonic 台式 CD, 盯着那台都布满灰尘了的便携 CD, 重新听 CD 的欲望一下就被重新点燃了。

Coldplay Viva La Vida 日版 FLAC 刚被还原成 CD

由于毕业时把大部分的 CD 留在了北京，身边的 CD 不多，又不想买 CD 了，所以还是选择自己下 FLAC APE 来自己刻，大学时期喜欢用三菱蓝盘，现在早已经停产，想到毕业的时候把那么多的蓝盘都送了人，真是后悔不已，现阶段性价比较高的就是莱德的音乐盘了，立马从网上买了一套 25 张。刻录方法继续用 Foobar2K 把 FLAC 和 APE 还原成 WAV, 然后用 Burnatonce 把 WAV 最终刻成 CD. 大学用这种方法刻了整整一 CD 袋，这种方法的音质个人觉得还是比较完美的。

至于播放器，房东这台台式 CD 非常出色，是款松下的老机子了，带音箱带功放，边玩 FM, 边听听，又回到大学了，呵呵，至于那台松下 CD820, 真是好久没听了，还被我借给别人大半年，CD Player 虽然慢慢停产，但它的意义永远在那里，无论怎样也不会卖了它（虽然现在它的二手价格竟然比我买新的时候还贵）。

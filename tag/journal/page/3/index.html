Friday, 24 October 2008

累，就一个字！

终于又是一个周末了，这几个月来，累！就一个字！

有曼联欧冠比赛的凌晨，闹钟响了，我也醒了，硬是不想起，想当年，只要是有曼联比赛的凌晨，根本不用闹钟，全凭念力在 2：30 分准时自然醒，而现在…… 天天重复着相同的工作，现在的湛江还算得上是夏季，搞的我身上到处都是毛病，鼻子上长疮，屁股上也长疮，腰也疼，眼睛天天都是 GUA 起的（四川话，不知道普通话怎么说），最神奇的就是每天下午下班前 5 点左右，头“准时”还是疼，晕，差不多连续半个月了，每天相同的症状，晚上回到家难得休息，可是上会网，浏览下 Feed, 玩几场 FM，踢几场实况，一下子就十点半了，以前十点半乃是我精神最佳的时候，可是现在，一到十点半，眼睛就睁不开了，哎！照理说，这么疲惫的身体晚上一定睡的很好，可是，倒在床上一睡着就开始做梦，一直做到天亮，就这样一复一日。我现在特别想休假，我今年还有一个 5 天的年休假和 20 天的探亲假没休，可是现在都年底了，一到年底活就多起来，看来是没希望了，这就是国企的休假制度！

以前读书最多就是为考试、升学发愁，可是从来没有现在这样累过，身体、身心双重劳累……

Sunday, 14 September 2008

广东话学习进度

虽然工作环境都是普通话，但通过广东地方台和几个香港台的电视轰炸和自己刻意学习，来广东这边工作一年，目前为止广东话进度为：

看懂广东话：75%
听懂广东话（比较书面的电视新闻、电视广告类）：60%
听懂广东话（口语较多的日常）：45%
说广东话（比较书面的电视广告类）：20%
说广东话（口语较多的日常）：10% 知道用词，发音，但音调始终不对，开不了口。

作为四川人学广东话其实有便利之处，有些发音相近或相同，广东话七个音中除了普通话的四个音，另有两个音四川话也有。继续努力！！

Saturday, 13 September 2008

重拾 CD

毕业以后，只买过一张 CD, 更别说听 CD 了，曾经在中国政法大学南门淘 CD 的激情早就没有了，最近玩 FM2008 热刺的时候，盯着电脑机箱上的刻录光驱，盯着床头那台房东的 Panasonic 台式 CD, 盯着那台都布满灰尘了的便携 CD, 重新听 CD 的欲望一下就被重新点燃了。

Coldplay Viva La Vida 日版 FLAC 刚被还原成 CD

由于毕业时把大部分的 CD 留在了北京，身边的 CD 不多，又不想买 CD 了，所以还是选择自己下 FLAC APE 来自己刻，大学时期喜欢用三菱蓝盘，现在早已经停产，想到毕业的时候把那么多的蓝盘都送了人，真是后悔不已，现阶段性价比较高的就是莱德的音乐盘了，立马从网上买了一套 25 张。刻录方法继续用 Foobar2K 把 FLAC 和 APE 还原成 WAV, 然后用 Burnatonce 把 WAV 最终刻成 CD. 大学用这种方法刻了整整一 CD 袋，这种方法的音质个人觉得还是比较完美的。

至于播放器，房东这台台式 CD 非常出色，是款松下的老机子了，带音箱带功放，边玩 FM, 边听听，又回到大学了，呵呵，至于那台松下 CD820, 真是好久没听了，还被我借给别人大半年，CD Player 虽然慢慢停产，但它的意义永远在那里，无论怎样也不会卖了它（虽然现在它的二手价格竟然比我买新的时候还贵）。

Monday, 28 July 2008

随便说两句

7 月初回到湛江后 Blog 就一直处于闲置状态，一是实在没什么可写的，二是回来后太忙。7 月份的湛江热的要命，太阳晒在身上尤如火烤，每天工作虽然有 4 台大空调吹着，但还是处于中暑状态。另一方面又要忙着找房子租。

经过半个月的寻找，终于在离公司较近的地方找到一个三室两厅的房子，个人非常满意，全新装修过的，家电家具基本为齐全，房租虽然在本地算是比较贵的，但和那些大城市相比也只是一个零头，住的舒服点也就得了。虽然工作一年了，但直到现在才算脱离了宿舍生活。其实也挺想念宿舍生活的，可以一起下楼踢球，一超联机魔兽 CS FM，一起玩实况，重要的是不要房租，水电什么都不要钱（9 月份要开始收了）。当然出来生活的优势还是相当明显的。搬出来后，陆陆续续地添加了一些家电，去电信局把电话宽带办理好，东拉西扯，终于有个家的样子了。

今天刚刚通网，以后只要无聊的时间多，Blog 的更新又会重新频繁起来，足球、英超、曼联、音乐、闲扯还会是我 Blog 的主题，现在能静下来上网了，说不定又要重新开始关注 Web 世界了。

这样下去，整个人将彻底麻木，沿着这条路不知道要走到何时、何地。

Sunday, 15 June 2008

第一次现场看国足，恐怕也是最后一次

难得遇上一次能现场看国家队比赛的机会，虽然对国足早已由爱生恨，但这种机会真的很少，加上从未去现场看过国家队的比赛，再加上这场比赛不怎么吸引人，球票价格跳水跳的深，于是和另一名同事从塘沽“远赴”奥体中心去看比赛了。

天津奥体中心同成都体育中心、北京工体比起来真是漂亮多了，无论外部造型还是内部造型，绝对五星级标准，国家队的表现真是太对不起这座球场了。我的位置在电视画面对面二层上，俯瞰全场，视觉效果非常不错，“音响效果”也很 NB, 球迷的吼叫声给人一种立体感觉，不像工体，给人一种看“坝坝电影”的感觉。尽管赛前球票经营惨淡，但“据目测”还是来了大约有 4 万人，主要分布在东西两侧。

尽管现场看球与电视机前看球的最大区别就是气氛不同，但整场比赛我也就激动了三次，一、奏国歌时，二、中国队进第一个球时，三、比赛快结束时我所在的位置下方的一群人高喊“国足解散”然后带动全场高喊时。前两次激动可以让你忘记所有中国足球的不好，后一次激动可以让你一下又回忆起所有中国足球的垃圾！最后几分钟，那帮 SB 一点斗志都没了，我们要急着赶车，提前离场了，远远的还听见巨大的“解散”的声音……

就这样吧，这是第一次现场看国足，恐怕也是最后一次……

Continue Reading]]> 继续沉浸在 Flickr 被 GFWed 的悲痛中，今天却发现了意外惊喜，巧克力工厂 的 Micheal 结合半亩塘闲话的日志，给出了挽救 Blog 上 Flickr 图片的方法。原来，GFW 封的只是域名，IP 没被封，所以可以把 URL 地址替换成 IP 形式就可以了，图片在 flickr 上一般被存储在 static.flickr.com, farm1.static.flickr.com 和 farm2.static.flickr.com 上，所以在数据库中把这三个地址替换成对应的 IP 地址就可以了。

方法: (via Michael)

以下的操作对象为 WordPress 的数据库，其它类型的 blog 请自行参照修改。

在 Myphpadmin 的 SQL 中分别输入：


UPDATE `wp_posts` SET `post_content` = REPLACE(`post_content`, 'farm1.static.flickr.com', '68.142.232.116');

UPDATE `wp_posts` SET `post_content` = REPLACE(`post_content`, 'farm2.static.flickr.com', '69.147.123.56');

UPDATE `wp_posts` SET `post_content` = REPLACE(`post_content`, 'static.flickr.com', '68.142.232.116');

替换完成。

UPDATE: iStef 为 WordPress 用户写了个插件，激活后，Flickr 的图片直接回来！不错不错（iStef 的文章里还有很多其它方法，也不错）。

]]> https://blog.nzh.me/my-life/flickr-seems-to-being-gfwed.html/feed 17 Picasa, Yupoo, Flickr https://blog.nzh.me/computer-and-internet/picasa-yupoo-flickr.html https://blog.nzh.me/computer-and-internet/picasa-yupoo-flickr.html#comments Sat, 31 Mar 2007 09:47:02 +0000 http://www.playmyself.com/computer-and-internet/picasa-yupoo-flickr.html 看了y0ungs写的我的图片存储新组合后，我也想写篇日志简单说一下网络相册储存服务和我是使用这些这些服务的情况。我只写我用到的三个：Picasa, Yupoo, Flickr，为什么没有另一个国内比较出名的巴巴变？一，它在我这速度超慢；二，登进去后，只要一操作就自动登出。（我RPWT？）

PicasaWeb出自Google之手，刚刚给免费用户容量升级到1G，Picasa功能简单，同Picasa软件、Blogger搭配最佳，我用它来存储私人照片，比如同学聚会照，男生宿舍自爆照等；只限于同学之间看，它功能简单，所以使用也简单，浏览起来也比Yupoo, Flickr方便。至于容量1G，也足够了，不够就再注册个Google 账号就行了。
我的链接：都说是私人相册了，说出来你们也看不了

Yupoo，我心中国内最好的图片存储：每月61M流量，节假日还有增量，其它什么都不限；界面清爽，导航明了，在改版后更加优秀，逐渐有了自己的界面特色；服务也多，比如彩信，还有个NB的功能就是可以从Flickr等相册直接搬家到Yupoo，（这个功能太XX了）。虽然Yupoo很优秀，但我只用它来传非原创图片（Images），我Blog里非原创图片都是传到Yupoo的。
我的链接：http://skyland.yupoo.com/，也没什么好看的，全是些杂图。

Flickr，我心中全球最好的图片存储，并付了$24升级到Pro，Pro用户用起来爽呆了，功能应有尽有，无所不有。Flickr开放了API，全世界支持Flickr的软件插件超多，Wordpress也有很多专门针对Flickr的插件。Flickr背后是Yahoo! 虽然前段时间Yahoo!做了一件不厚道的事，强行让以前在Flickr申请的ID换成Yahoo! ID，但我本来就是Yahoo!ID登录的Flickr，所以我也不是很生气-_-。我所有的原创图片，和照片（Photo）通通上传到Flickr。
我的链接：http://www.flickr.com/photos/skyland_mu

Flickr不可否认的比Yupoo强大，尽管Yupoo已经慢慢的模仿出了很多功能，并开始有自己的特色，但还是有两大弱点让我不能用它的NB功能让我从Flickr搬家到Yupoo，一是这篇文章里说到的Yupoo生存问题，巴巴变现在已经有了收费VIP服务，但Yupoo仍然在搞技术开发，没任何收费服务，至今还没听说有什么注资融资消息，二就是缩略图的质量问题：两者都可以输出不同的尺寸（Size），像要引用大图片到论坛Blog，一般选择较小尺寸的输出，这样一来，Yupoo缩略图输出的质量劣势就明显了，两者都有账号的可以试试同一张图片，输出同样的Size，Flickr的输出明显锐化的真实清晰些，Yupoo的简直惨不忍睹，特别是PNG。不过，我两者的原始Size输出图片质量一样，和你本地图片的MD5值都一样，保证了图片的原汗原味（虽然Yupoo的PNG图片格式会改成JPG，但其实是一样的，只是个后缀问题）

以上是Flickr PRO和Yupoo的对比，免费Flickr有很多限制，比如你只能看到你最近上传的200张（以前的还在，外引也不失效，只是看不到），Set只能设置三个等，所以如果你不是特别喜欢Flickr、不是Flickr Pro的话，Yupoo是你最佳选择，相信它也没那么快挂掉。

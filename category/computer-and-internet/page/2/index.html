开玩 Windows Phone 7

从 WP7 手机开发之时，其中的 XBOX LIVE 联动功能就吸引着我，WP7 系统出来快一年了，虽然现在还没有官方中文，但还是禁不住诱惑，入了一台三星 Omnia 7，现在 WP7 的手机不多，所以选择余地也小，减少纠结的痛苦，而且现在的 WP7 手机已经到了一个很好的价位。

简单感觉：
手机很大，4 寸的屏幕，一个手操作不太容易，加上电容屏不能用指甲点，还是两个手操作来好些，由于 WP7 的手机硬件很严格，所以机子也很流畅。系统没中文，但显示中文没问题，稍微 GEEK 点的同志应该没任何语言障碍，输入中文已有第三方软件，所以语言不是问题。

手机系统界面是让人耳目一新的 METRO UI，在以后的 WINDOW 8 和今年秋天将要更新的 XBOX DASHBOARD 都将统一成 METRO UI 的风格。图标大，字体大，很简洁，颜色搭配也舒服，看习惯了塞班的 9 宫格，和 Android 的 N 宫格，METRO UI 绝对让你充满新鲜感。

手机一到手马上连接上 Zune 更新到最新版本，想都没有想过要越狱，因为越狱了，游戏就没有 XBOX LIVE 功能了，没成就解了，而且我看了下 MARKETPLACE，免费应用基本够，就是游戏大作是收费但也不贵，（我的 LIVE 是港服，开始看价格吓了我一跳，后面才反应过来那是 HKD，不是 USD）。 WP7 手机必须靠 Zune，更新、同步什么的都靠 PC 端实现，Zune 还是很好用的，与 WP7 手机联动感觉比苹果设备与 iTunes 方便。

XBOX LIVE:
WP7 手机对我最大的吸引就是与 XBOX LIVE, XBOX 360 的互动，用你 XBOX LIVE ID 作为手机的主 ID，玩有成就系统的游戏，就可以享受在手机上玩游戏还有成就解的“快感”了，试了试一个免费的 FLOWERZ，我用手机玩时，在 XBOX 360 上的好友能看到我在玩游戏。手机游戏成就解除的声音和在主机上一样，当我解开一个成就时，那熟悉的声音跳出来，那个亲切啊。游戏成就与记录一并加入你的 XBOX LIVE ID 设定档。

另外，还可以在 WP7 手机上查看成就，编辑小人，收发消息，等 MANGO 系统出来后，GAME HUB 将有更多功能，听说还可以与 KINECT 互动。

New Mouse: Logitech G5

去年“上当”买了支 Rapoo 的垃圾无线鼠标，忍受了一年终于忍不下去了，再次出手，整了个罗技 G5. 这个系列鼠标是我们公司新地面系统的专配鼠标，用起来相当舒服，一直想办法“偷”一个回来用，但始终下不了手，无奈只得自淘腰包买一个了，不要让人误会是我从公司“偷”的就好了。由于 G5 是我体会了无数次的，所以这次也就不可能像上一次那样后悔了。其实这是一款游戏鼠标，虽然我也玩 CS, 魔兽之类的对鼠标要求较高的游戏，但我这个水平还不至于要专门配游戏鼠标，这个鼠标我特别喜欢它的磨沙处理，和握起来的饱满感觉，而且在公司经常用，再加上也符合我的审美观，就这样了，呵呵。虽然装上驱动功能更多，但我感觉都用不上，直接插上就用。唯一的缺陷就是滚轮键向下按时不太“有感觉”，在 Fx 浏览器下点击链接新开标签页时不太顺手。

五笔使用者终于有的选择了……

小时候学过一阵五笔，后来大概有十年没用了，大一寒假，老爸命令我学习五笔，结果依靠小时候的记忆，通过一款软件，半天之内就把五笔掌握完全了，通过 QQ 聊天，论坛灌水几天时间就熟悉了，从此开始了五笔生涯。

后来不久认识极点五笔，从此就再也没用过第二款五笔输入法，这几年来，一直在网上看到搜狗的拼音输入法口碑很好，而且拼音输入法市场真是百花齐放，连 Google 也来凑热闹，可是五笔市场除了极点一家，就没第二家优秀稍微好用点的，今天却发现了搜狗也出了一款五笔输入法，介于它在拼音输入法中良好的口碑，怎么也得试一下。初步感受就是，还是没极点好用。

一，字库，我极点五笔一直用的 4.1 版本，没有更新，字库也是“窝子词库”加入我一些自造的词，搜狗五笔虽然有很多“很新颖”“很网络化”的词库，但像“终于”“英文”“短信”这些词都没有，我发现一个规律，凡是如果一条词语的代码同时也是一个字的代码时，这个词就没有，这样造成很多常用词组不能联想，还得按 Shift Ctrl Z 自造！能直接打出曼联、阿森纳、阿仙奴、利物浦等国、粤译名，Chelsea 的却只能直接打出粤语译名“车路士”，这可能是网络反馈的词组，因为“车子”这个词已经深入人心，有趣。另外大多球队球星名字都能直接打出，只可惜我平常不喜欢用中文译名。-_-
二，不能调整为 GBK 字库，GBK 繁体，Big5 繁体，不知道是不是我没发现还是没有，而极点五笔几个快捷键就可以切换。
三，不支持首字自动上屏，有时要连打几个字，如果第一个字是四码重码，不会自动上屏，这样造成输入后面字的码后输出的就全是字母，这一点相当不爽。

上面这段文字是在搜狗五笔下打的，准备继续回去用极点五笔，等搜狗的升级版。

挽救 Blog 上的 Flickr 图片！

继续沉浸在 Flickr 被 GFWed 的悲痛中，今天却发现了意外惊喜，巧克力工厂 的 Micheal 结合半亩塘闲话的日志，给出了挽救 Blog 上 Flickr 图片的方法。原来，GFW 封的只是域名，IP 没被封，所以可以把 URL 地址替换成 IP 形式就可以了，图片在 flickr 上一般被存储在 static.flickr.com, farm1.static.flickr.com 和 farm2.static.flickr.com 上，所以在数据库中把这三个地址替换成对应的 IP 地址就可以了。

方法: (via Michael)

以下的操作对象为 WordPress 的数据库，其它类型的 blog 请自行参照修改。

在 Myphpadmin 的 SQL 中分别输入：


UPDATE `wp_posts` SET `post_content` = REPLACE(`post_content`, 'farm1.static.flickr.com', '68.142.232.116');

UPDATE `wp_posts` SET `post_content` = REPLACE(`post_content`, 'farm2.static.flickr.com', '69.147.123.56');

UPDATE `wp_posts` SET `post_content` = REPLACE(`post_content`, 'static.flickr.com', '68.142.232.116');

替换完成。

UPDATE: iStef 为 WordPress 用户写了个插件，激活后，Flickr 的图片直接回来！不错不错（iStef 的文章里还有很多其它方法，也不错）。

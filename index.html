总结推荐科学服务

距离上次推荐的科学服务已经过去好几年了，这几年国际线路对中国大陆的变化也改变了很多，比如之前的神线路电信 CN2 GT 已经彻底拉跨，GT GIA 崛起，在用了一段时间的机场服务后，最近又开始疯狂折腾起了 VPS，现在的标准不仅要线路好，还想着能完全解锁 Netflix 服务。

但是，越来越觉得折腾这些东西太浪费时间，如果接下来国内国际网络没有重大调整，我基本不会再折腾 VPS 服务器了，以下是我折腾一翻后沉淀下来一直在续费使用的，使用的非常满意，所以不做详细评测与描述。

MoeCloud HK BGP 本地带宽大，国际互联性良好，配合广州移动中转11ms延迟，平常主力使用；

MAXKVM Singapore 可解锁 Netflix 非自制，配合广州移动中转43ms延迟，看Netflix主力使用；

WikiHost SCP转发，含有 IPLC 转发节点，充值不同节点按流量计费，主要使用广州移动节点拉上面两台鸡；

SaltyFish 德国 CN2 GIA，相比美国 GIA 使用人数更少，而且可以用来中转一大批欧洲高性价比小鸡、服务器等，访问欧洲流媒体使用；

Olink Cloud 美国9929线路，使用感受超CN2 GIA，YouTube 等流媒体主力使用；

Blog 全面静态化，部署到 Vercel

虽然万年不更新了，但这个 Blog 一直放在阿里云新加坡上，阿里云新手套餐 2.0 没有以后，又折腾了一圈，感觉太累了，反正万年不更新，就索性全面静态化，直接 Push 到 Github Pages，再用 Vercel 自动 Deploy 吧…

用了十几年的 WordPress 也不太好转成其他静态化 Blog 程序，就干脆在本地 Win10 Linux 子系统搭建好后端，更新什么的就在后端，再生成全站静态化，最后推至 Vercel，加上 Vercel 的 CDN 是 Google Cloud，中国访问基本上很快，省了很多麻烦，尤其是我这种基本不更新内容的人，就当是在互联网上留个痕迹吧！

其实呢，写这一遍，也只是我测试一下整个流程而已，并不是想更新！

Sunday, 29 January 2017

为什么我用 Spotify

  • 纯粹 无论 macOS 桌面版， iOS 版都只关注音乐，没有国内网易云音乐等乱七八糟的东西，更没有我最讨厌的 App 启动广告，至于评论，现在网易云下面的评论还能看么？各种抄的段子，编的故事，装逼的矫情…
  • 算法推荐 Spotify 的 Discover Weekly，Release Radar 无与伦比，每周都能从算法推荐的歌单中找到心仪的歌曲… 更有 Pitchfolk 和各国 Topsify 生成的歌单，比起网易那些文艺小青年的歌单高大上多了。（或者说更符合我的口味）
  • 曲库 欧美基本齐全，更有大量北欧小众 Indie 音乐可听，用香港区也可以大体照顾到华语曲库，至于实在没有版权的专辑，Spotify 高级会员根本不是问题，从其他渠道下载到本地，再导入本地曲库即可同步到各个客户端听…
  • 全平台 Spotify 与 PlayStation Network 融合(PlayStaion Music)，直接打通我平常生活 90% 的平台：电脑、手机、PS4… 在 PS4 上登录 Spotify 更能等到来自游戏界的原声带音乐、各种游戏主题音乐，并且可以在玩游戏的时候后台播放，在玩各种日式 RPG 需要各种刷素材的时候打开做为背景播放，刷起来更加轻松。并且 Spotify 的多平台是可以相互控制的，不像其他音乐的全平台除了数据外其他是独立的…
  • 来自瑞典 来自有 伊布拉希莫维奇、沃尔沃、Saab、哈苏、宜家的国度…

说到 Spotify，最大的阻碍就是墙与付款问题吧，墙的问题这稍微 Geek 点的小白现在都不是问题了吧，付款问题的确是个问题，要当地信用卡，当地 PayPal，国内发行的都不行，解决办法是：淘宝买 Gift Card；有 PSN 账户的绑定到自己的 PSN 通过 PSN 就可以用国内银联卡或信用卡支付了；加入网友的家庭套餐，只要「家长」可以支付，5 个人就可以享用家庭计划了，香港区一个人一月大概 15RMB. 你可以先用免费版 Spotify，心动了的话想升高级版，付费方式总会想到办法的…

我怀念的是碧海蓝天

这半年来，一切还好，对做出的决定没后悔过，相反某些事情还让我庆幸做出了决定，无论如何，自己的决定自己负责…

然而冬天来了，总有一样东西还是让我无法忘记，就是以前生活的那座城市那温暖的冬天，没有口罩、没有雾霾，那片碧海蓝天！

上一次长期居住在冬天四川，那是 2002 年了，之后上大学工作都在外面，只有过年才回家，那时候的中国那有什么雾霾的概念，PM2.5 也就是最近几年才开始关注的，一直以来，每当在新闻上看到哪哪哪 PM2.5 爆表，我只是微微一笑。

而当我自己过着这种成日不见太阳，雾霾重重的生活时，无论从身体还是身心都是难受的，之前写的那种「压抑感」更重了，我可以忍受重污染，但我受不了这种压抑，还有就是重污染空气中的那股说不出来来自哪里的怪味道…

以前，我内心总是觉得那些天天抱怨空气不好的大城市的人太傲骄，现在亲身经历后，我错了…
